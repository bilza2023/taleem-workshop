“ہم روزانہ numbers استعمال کرتے ہیں — counting کے لیے، measurements کے لیے…

لیکن ایک سوال ہے…
**کیا سارے numbers ایک جیسے ہوتے ہیں؟**

حقیقت یہ ہے کہ numbers وقت کے ساتھ evolve ہوئے ہیں۔
جیسے جیسے لوگوں کو نئی ضرورتیں پیش آئیں… ویسے ویسے نئے types کے numbers سامنے آتے گئے۔

چلیں اس journey کو step by step سمجھتے ہیں۔

سب سے پہلے آتے ہیں **Natural Numbers**۔
یہ وہ numbers ہیں جو ہم counting کے لیے استعمال کرتے ہیں —
1, 2, 3, 4…
یعنی جب ہم چیزیں گنتے ہیں، ہم Natural Numbers استعمال کر رہے ہوتے ہیں۔

لیکن پھر ایک مسئلہ آیا…

اگر کچھ ہو ہی نہ تو اسے کیسے ظاہر کریں؟

اسی ضرورت کی وجہ سے **zero** شامل کیا گیا،
اور ہمیں ملے **Whole Numbers** —
0, 1, 2, 3…

پھر وقت کے ساتھ ایک اور ضرورت پیش آئی…

لوگوں کو profit اور loss کو ظاہر کرنا تھا،
temperature بھی zero سے نیچے جا سکتا تھا…

اس مسئلے کو حل کرنے کے لیے **negative numbers** آئے،
اور یہ سب مل کر بناتے ہیں **Integers** —
…, -3, -2, -1, 0, 1, 2, 3…

اب تک سب کچھ ٹھیک لگتا ہے…
لیکن ایک اور مسئلہ باقی تھا…

اگر کوئی quantity پوری نہ ہو؟
جیسے half — 1/2 — یا 2.5…

جب لوگوں نے چیزوں کو حصوں میں تقسیم کرنا شروع کیا،
تو ہمیں ضرورت پڑی **fractions** کی۔

اسی سے بنتے ہیں **Rational Numbers** —
وہ تمام numbers جو fraction کی form میں لکھے جا سکتے ہیں،
جیسے 1/2, -3/4, 2.5…
یہاں تک کہ integers بھی، کیونکہ 3 کو ہم 3/1 لکھ سکتے ہیں۔

اب لگتا ہے کہ ہم نے سارے numbers cover کر لیے…

لیکن یہاں ایک surprising بات آتی ہے…

کچھ numbers ایسے ہوتے ہیں جو **کبھی بھی fraction میں نہیں لکھے جا سکتے**۔
ان کی decimal form نہ ختم ہوتی ہے… اور نہ ہی repeat ہوتی ہے۔

جیسے √2… یا π…

انہیں کہتے ہیں **Irrational Numbers**۔

اور آخر میں —
جب ہم **Rational Numbers** اور **Irrational Numbers** کو combine کرتے ہیں…

تو ہمیں ملتا ہے سب سے بڑا set —
**Real Numbers**۔

یعنی وہ تمام numbers جو number line پر represent کیے جا سکتے ہیں۔

اگر ہم پوری picture کو دیکھیں…
تو ایک بات بالکل clear ہو جاتی ہے:

**ہر نئی number family… پچھلی family کی limitation کو solve کرتی ہے۔**

اور اسی طرح… step by step…
پورا number system بنتا ہے۔”
